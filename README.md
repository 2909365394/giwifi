# giwifi
校园网一键认证

### Padavan使用方法

1、先使用ssh登陆路由器   
2、安装opkg  
```
opkg.sh
```
3、安装python3   
```
opkg install python3   
```
4、安装pip3
```
opkg install python3-pip
```
5、安装依赖库**requests**
```
pip3 install requests
```
6、下载giwifi.py源码
```
wget http://mgh234/giwifi.py
```
7、打开Padavan管理界面
在<kbd>自定义设置</kbd>---><kbd>脚本</kbd>---><kbd>自定义脚本0(功能配置)</kbd>
